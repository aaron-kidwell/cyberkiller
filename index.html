WHY TEAMS RUN IT

👑 Automated King of the Hill

Flip any box to KOTH mode and the platform tracks the throne itself. Get root, write your handle to the flag file, hold it. Points tick up live. No koth client to run.

🎯 Any box, in seconds

Add a target by registry reference or upload a docker save tarball. Flags auto-plant, even on images that know nothing about CyberKiller.

🏴 Capture by handle

Prove a capture by writing your hub handle into the flag file. The server reads it and awards you automatically. First blood, kills, live board.

🧡 Real chains

The bundled MERIDIAN scenario is a designed 10-box pivot: breach the DMZ, loot creds, hop inward, root the network.

🔐 Private & yours

Your players, your network, your data. Run it on a laptop, a VPS, or an air-gapped lab.

🚀 One command

./deploy.sh builds everything and prints the hub URL + an invite code. Tear it down or wipe it just as fast.
